if-else/if

2017-04-14

最近在OJ上刷题，总是会“莫名其妙”的出现许多错误，截止今天再次莫名出错，着实不能再忍受，仔细翻阅与之前犯错的对比发现，对于if-else/if理解的竟然有偏差，直接上代码

for i in range(10):
    if i<5:
         print "A",
    print "B",

A B A B A B A B A B B B B B B

for i in range(10):
    if i<5:
        print "A",
    else:
        print "B",

A A A A A B B B B B

重新翻阅基础知识，if/else 单词执行只执行其中一步，同try/except

而if+，之后的语句则是不管if执行与否，其后程序必执行

很遗憾，这么简单的问题啊，竟然稀里糊涂的用了这么久（基础啊基础！）
