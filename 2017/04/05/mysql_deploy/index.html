MySQL数据库如何实现远程登录

1.修改配置文件

/etc/mysql
找到my.cnf，不清楚mysql配置文件编码的话，建议使用
vi /etc/mysql/my.cnf
i到bind-address 127.0.0.1
注释掉这句话：# bind-address 127.0.0.1
：wq保存并退出，如果出现问题，则使用
：w !sudo tee %

2.修改mysql host

mysql> use mysql
mysql> update user set Host=’%’ where User=’root’
mysql> GRANT ALL PRIVILEGES ON . TO ‘myuser’@’%’ IDENTIFIED BY ‘mypwd’ WITH GRANT OPTION
mysql> FLUSH PRIVILEGES

3.重启mysql

/etc/init.d/mysql stop
/etc/init.d/mysql start
